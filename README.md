## 🤖 Built with AI / 用 AI 构建

> AI 不是补全工具，而是协作者：它扩张我的产出半径，让一个人也能交付带 AI、带文档、带测试、带 CI 的完整系统。
>
> **I don't just write code — I ship ideas with AI.**<br/>
> AI isn't autocomplete; it's a collaborator. It widens my radius of output, so one builder can deliver a complete system: AI inside, docs, tests, and CI included.
